Hi! fccBot, at your service!

Ever wish you could get information about a freeCodeCamp challenge? Right from Slack? Now you can. Whether you're looking for a linked resource, are curious about a project objective, or just want to double-check the user stories for whatever project you're working on, fccBot has your back. Just install the slack app, then summon fccBot whenever you need help. It's that simple.

fccBot in Action

To get information from fccBot, all you have to do is ask. He's (mostly) happy to help.

Where can I learn more?

fccBot was created by members of the Chingu Cohorts - student programmers learning together. The fccBot is not part of the freeCodeCamp project and was not created or endorsed by freeCodeCamp - but we like to think they'd approve.

To learn more about fccBot, check out the GitHub repository for the fccBot project.
